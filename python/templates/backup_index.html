Order To Chois

面倒で時間ばかりかかる発注作業を自動化しましょう👍
このツールでは、CHOISの一括発注に必要なCSVファイルを作成できます。

過去に発注した医薬品を95%の確率でカバーするように設計されています。
欠品がないことを重視した設計のため、月初発注として使うのがオススメ👨‍🏫

Order To Chois の仕組み

Order To Choisでは、記述統計に基づいて出庫量を予測しています。
まず、これまでの出庫量のバラつきを計算し、95%の確率で出庫量を推測します。
予測される最大の出庫量から、現在の在庫数を引いた数値を充たすように、発注数を計算します。
詳しくは、需要予測による在庫管理や、母平均の推定精度の指標【統計学・統計解析講義基礎】をご覧ください。

途中から再開したい方はコチラ

発注途中のファイルがある場合は、こちらから再開できます。
現在庫は自動更新されないので、できるだけ早めに発注を完了してください。

発注を再開する

一括発注用ファイルを作成する

  1. Step 1

    過去1年分の入庫データ

    1. 「入庫・出庫」タブを開く
    2. 入庫・出庫伝票検索をクリック
    3. 入庫区分から、
      「仕入」「仕入（手書）」のみに✅
    4. 伝票日付を現在月を含まないように1年分設定
      * 今が2022/5なら、
      2021/5/01～2022/4/30と設定
    5. 「ファイル出力」から自分がわかるように名前を付けてCSVを保存
  2. Step 2

    5ヵ月分の出庫データ

    1. 「入庫・出庫」タブを開く
    2. 出庫区分から、「処方」のみに✅
    3. 伝票日付を現在月を含まないように5～6か月分設定
      * 入庫と同じです。
      CHOISが1回に9999行しか出力できないので、
      限界に達していたら月数を減らしてください
    4. 「ファイル出力」から自分がわかるように名前を付けてCSVを保存
  3. Step 3

    医薬品集

    1. 「在庫・棚卸」タブを開く
    2. 在庫一覧をクリック
    3. 医薬品集ボタンをクリック

作成には10～20秒程度の時間がかかります。

投薬前にポチっとするか、コーヒーを飲むなどしてお待ちください🙇

🔥 注意

一括発注用ファイルはエクセルで編集するとJANコードが自動で変更されてしまいます。
開かずにこのツールで作成したファイルをCHOISへ取り込んでから「行削除」を使って削除してください。
